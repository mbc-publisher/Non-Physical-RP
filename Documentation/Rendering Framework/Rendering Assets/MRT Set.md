<h1 align="center">MRT Set</h1>

<h3 align="center">Description</h3>

<div align="center">Describes a set of render targets that will be rendered at the same time to save on rendering resources.</div>

<h3 align="center">Parameters</h3>

| Name | Type | Choices | Description |
| :---- | :---- | :---- | :---- |
| Colors | Array / List | Any Non-Depth render targets that are not already in the set. |  |
| Enable Depth Override | Boolean |  | Enable the writhing of custom values to the depth buffer. |
| Depth Render Target | Enum | Any Depth render target |  |
