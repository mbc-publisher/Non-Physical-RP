<h1 align="center">Clear</h1>

<h3 align="center">Description</h3>

<div align="center">Clear chosen render target(s). Can be used to clear a render target before rendering something else on it. Can also be used to delay the automatic clearing of render targets.</div>

<h3 align="center">Parameters</h3>

<div align="center">
  
| Name | Type | Choices | Description |
| :---- | :---- | ----- | :---- |
| Color RTs | Array / List | Any Non-Depth Render Target | The Non-Depth Render Target(s) that will get cleared. |
| Depth RTs | Array / List | Any Depth Render Target | The Depth Render Target(s) that will get cleared. |

</div>
