<h1 align="center">Luminance</h1>

<h3 align="center">Description</h3>

<div align="center">Get the luminance of a color value.</div>

<h3 align="center">Ports</h3>

<div align="center">
  
| Name | Direction | Type | Binding | Description |
| :---- | :---- | :---- | :---- | :---- |
| In | **Input** | Vector 3 | None | Color value. |
| Out | **Output** | Float | None | Luminance of the color. |

</div>

<h3 align="center">Options</h3>

<div align="center">
  
| Name | Type | Choices | Description |
| :---- | :---- | :---- | :---- |
| Type | Enum | •True<br> •Visual | **True:** Average of the RGB channels.<br><br> **Visual:** The RGB channels are weighted according to the human perception of color. |

</div>
